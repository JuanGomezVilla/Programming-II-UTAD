## Compilación y ejecución
- Compilación del código:
    ```bash
    compile filename.c 
    ```
    > El archivo resultante es un ejecutable de nombre _ejercicio.exe_
- Compilación y ejecución del código:
    ```bash
    run filename.c
    ```
    > El programa compilará el código y luego lo ejecutará
